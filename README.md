# My website

My personal website, built with [Next.js 13](https://beta.nextjs.org/).

## Development

1. Install dependencies

```bash
pnpm install
```

2. Run the development server

```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
