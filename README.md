# My website

My personal website, built with [Sveltekit](https://svelte.dev/).

## Development

1. Install dependencies

```bash
pnpm install
```

2. Run the development server

```bash
pnpm dev
```

3. Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
